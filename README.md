# Job Board Platform

A backend-powered Job Board Platform built with **Node.js**, **Express.js**, and **MongoDB** that connects employers with job seekers. Employers can create and manage job listings, while candidates can search for jobs, upload resumes, apply for jobs, and track their applications.

## Features

- User authentication and authorization
- Role-based access (Employer & Candidate)
- Create, update, and delete job listings
- Search and filter available jobs
- Upload resumes
- Apply for jobs
- Track application status
- Employer notifications
- Secure RESTful API

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
- Multer

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Jed-Emelle/CodeAlpha_Job_Board_Platform
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Create a `.env` file and add the required environment variables.

4. Start the development server
   ```bash
   npm run dev
   ```
